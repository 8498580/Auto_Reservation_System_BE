## 前言

​	目前，包括[BIT](http://seat.lib.bit.edu.cn)、[CSU](http://libzw.csu.edu.cn)、HLJU在内的诸多高校的图书馆都在使用盛卡科技公司开发的图书馆座位预约系统，这个仓库旨在辅助各位同学更方便地使用预约系统预约座位，不可用于非法用途。

​	目前配置文件只支持BIT图书馆部分席位预约，欢迎大家共同维护或提Issues。
